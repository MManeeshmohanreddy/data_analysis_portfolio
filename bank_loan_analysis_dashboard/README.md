🏦 Bank Loan Analysis Dashboard

## Overview
Analyzed 38,000+ bank loan records using SQL and Power BI to identify
high-risk borrowers and track financial KPIs across a 3-page dashboard.

## Problem Statement
How can banks identify high-risk loans early and improve reporting
efficiency for stakeholders?

## Dataset
- Records: 38,000+ loan entries
- Source: Bank internal loan data (anonymized)

## Tools Used
- SQL Server (CTEs, aggregations)
- Power BI + DAX (MTD, MoM measures)
- Excel (data cleaning)

## Key Findings
- High DTI ratio strongly correlates with loan default
- Regional clusters show higher risk concentration in specific areas
- MoM trends revealed a 12% rise in high-risk approvals
